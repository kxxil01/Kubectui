Kubectui is a Rust 2024 terminal UI for Kubernetes built with ratatui, crossterm, tokio, kube, and k8s-openapi. It provides many Kubernetes resource views plus detail overlays for YAML, events, logs, port-forwarding, scaling, probes, and related actions. Rough structure: src/main.rs orchestrates runtime, async refresh, and detail fetching; src/app.rs holds app state and input-driven state transitions; src/ui/views and src/ui/components render ratatui UI pieces; tests live under tests/.
